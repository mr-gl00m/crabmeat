The name **"CrabMeat"** and the **CrabMeat butter-claw logo** are trademarks of
Nathan Seals. They are **not** licensed under the Apache 2.0 license. Section 6
of Apache 2.0 explicitly withholds trademark rights; this file states the policy
that fills that gap.

## What this means

- **The code is open. The brand is not.** You can fork CrabMeat, build on it,
  and ship your own version under the Apache 2.0 terms.
- **Forks must rename.** A derivative or modified project may not call itself
  "CrabMeat," use the CrabMeat™ name or logo, or present itself in a way that
  implies it is the official CrabMeat project. Pick your own name and logo.
- **Nominative use is fine.** You may truthfully state that your project is
  "built on CrabMeat," "compatible with CrabMeat," or "a fork of CrabMeat";
  that is ordinary descriptive reference, not a brand claim.
- **Unmodified redistribution** of official CrabMeat releases may keep the
  name and logo, since it is still the official project.

## Status

"CrabMeat™" is used as an unregistered trademark. The ™ symbol asserts a
common-law trademark claim and requires no registration.

## Why

Keeping the name distinct from the code lets users trust that something called
"CrabMeat" is the project they think it is. Open code with a protected name is
a deliberate, well-established pattern.
